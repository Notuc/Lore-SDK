# Lore

> AI that knows your world.

Lore is an open source AI library for game developers. It adds intelligent NPC dialogue, quest generation, combat AI, economy systems, and more — powered by local models or cloud AI, with zero lock-in.

---

## Features

- **NPC Dialogue** — Context-aware conversations that remember players, react to game state, and stay in character
- **Local or cloud AI** — Run models locally with Ollama or connect to Anthropic Claude. Switch with one config line
- **Unity & Unreal support** — Native C# package for Unity, C++ plugin with Blueprint nodes for Unreal
- **Plug and play** — Drop a component onto your NPC, fill in a name and personality, done
- **Shared memory** — All modules share NPC state. 
- **Modular** — Include only what you need. No module depends on another
